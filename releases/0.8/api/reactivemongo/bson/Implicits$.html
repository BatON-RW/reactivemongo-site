Implicits

object Implicits

Implicits for appending a mix of BSONValue, Option[BSONValue] in BSONArrays, of (String, BSONValue) and (String, Option[BSONValue]) in BSONDocuments. Example:

case class CappedOptions(
  size: Long,
  maxDocuments: Option[Int] = None
) {
  def toDocument = BSONDocument(
    "capped" -> BSONBoolean(true),
    "size" -> BSONLong(size),
    "max" -> maxDocuments.map(max => BSONLong(max)))
}
