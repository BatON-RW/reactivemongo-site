TraversableBSONArray

class TraversableBSONArray extends TraversableBSONStructure[Int] with BSONArray

An array reader. It will give the value matching the index, if it exists.

This reader is lazy: it deserializes values only when required. Moreover, it memoizes the already deserialized values, to avoid n computations.
