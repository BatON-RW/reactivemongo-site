reactivemongo.api.gridfs

GridFS

case class GridFS(db: DB[Collection] with DBMetaCommands, prefix: String = "fs") extends Product with Serializable

A helper class to make GridFS queries.

GridFS creates two collections, files and chunks, that store respectively the files metadata ant their chunks. These collections are prefixed by a customizable name (usually "fs") followed by a dot character.

So, if the database is "media" and the GridFS prefix is "photos", the two collections are:

  • media.photos.files
  • media.photos.chunks
db

The database where the GridFS collections are.

prefix

The prefix of this GridFS. Defaults to "fs".
