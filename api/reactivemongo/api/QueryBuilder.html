QueryBuilder

case class QueryBuilder(queryDoc: Option[BSONDocument] = scala.None, sortDoc: Option[BSONDocument] = scala.None, hintDoc: Option[BSONDocument] = scala.None, projectionDoc: Option[BSONDocument] = scala.None, explainFlag: Boolean = false, snapshotFlag: Boolean = false, commentString: Option[String] = scala.None) extends Product with Serializable

A helper to make query documents.

You may use the methods to set the fields of this class, or set them directly. Reading the documentation about advanced queries may be very useful.

queryDoc

The query itself (actually the selector)

sortDoc

The sorting document.

hintDoc

The index hint document.

explainFlag

The explain mode setter.

snapshotFlag

The snapshot mode setter.

commentString

A comment about this query.
