TraversableBSONStructure

sealed trait TraversableBSONStructure[Key] extends BSONStructure

A structure reader. It will give the value matching the given key of type Key, if it exists.

This reader is lazy: it deserializes values only when required. Moreover, it memoizes the already deserialized values, to avoid n computations.

Key

The type of the keys of this structure.
